# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 0.1.0 | Yes |

## Reporting a Vulnerability

Please do not post credentials, private account details, or sync data in a public issue.

For security-sensitive reports, contact the repository owner privately or use GitHub's private vulnerability reporting if it is enabled for this repository. Include the platform, release file, and enough reproduction detail to understand the issue.
