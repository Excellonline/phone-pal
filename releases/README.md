# PhonePal Releases

This folder contains published PhonePal builds.

## Android

- `android/PhonePal-Android-0.1.0.apk`: install directly on an Android phone.
- `android/PhonePal-Android-0.1.0.aab`: app bundle build for store workflows.

## Windows

- `windows/PhonePal-Windows-0.1.0/`: complete Windows desktop app bundle.
- Run `windows/PhonePal-Windows-0.1.0/phonepal.exe`.
- Keep every file in the folder together. The executable depends on the bundled DLLs, Flutter runtime files, and `data/` directory.
